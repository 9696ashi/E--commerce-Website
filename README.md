# 🛒 Amazon Clone

A simple Amazon clone built using **HTML, CSS, and JavaScript**.
This project includes basic features like user authentication, product display, and cart functionality.

---

## 🚀 Features

* 🔐 User Registration & Login (using SheetDB API)
* 🏠 Home Page with products
* 📦 Product Details Page
* 🛒 Add to Cart functionality
* 💾 Data stored using API (SheetDB)

---


## ⚙️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* SheetDB API (for backend/database)

---

## 🔑 How It Works

1. User registers using the Register page
2. Data is stored in Google Sheets via SheetDB API
3. User logs in using the same credentials
4. Products are displayed on the home page
5. User can add items to cart

---

## 🔗 API Used

* SheetDB API

```
https://sheetdb.io/api/v1/your_api_id
```

---

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Make sure internet connection is active (for API)
